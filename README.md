# Sales Prediction using Machine Learning

## Project Overview

This project predicts future sales based on advertising spending across TV, Radio, and Newspaper platforms using Machine Learning techniques.

---

## Machine Learning Model

* Linear Regression

---

## Features

* Data preprocessing
* Data visualization
* Sales forecasting
* Model evaluation
* Advertising impact analysis

---

## Dataset

The dataset contains advertising spending data and corresponding sales values.

---

## How to Run

1. Install required libraries
2. Open `sales_prediction.ipynb`
3. Run notebook cells step by step

---

## Conclusion

The project successfully predicted sales using regression techniques and demonstrated how advertising spending impacts sales performance.
